# Question 3 — Graph Learning with Pymanopt on the Sphere Manifold

## Problem

Solve the graph learning problem on the Sphere manifold:

$$\min_{M \in \mathbb{R}^{(n-1) \times k}} \operatorname{tr}(X^\top (Q_\perp M)(Q_\perp M)^\top X) \quad \text{s.t.} \quad \operatorname{vec}(M) \in S^{(n-1)k-1}$$

for each color channel $c \in \{R, G, B\}$, using both **SteepestDescent** and **TrustRegions** optimizers over 100 iterations, with $k = n-1$ and $k = 30$.

## Implementation

### Cost function simplification

Rather than computing $Q_\perp M$ at every iteration, we precompute:

$$S = Q_\perp^\top X X^\top Q_\perp \in \mathbb{R}^{(n-1) \times (n-1)}$$

The cost then reduces to:

$$f(M) = \operatorname{tr}(M^\top S M) = \sum_{i,j} M_{ij} (SM)_{ij}$$

### Derivatives

- **Euclidean gradient:** $\nabla f(M) = 2SM$
- **Euclidean Hessian:** $\nabla^2 f(M)[V] = 2SV$

Pymanopt internally converts these to Riemannian gradient/Hessian via projection onto the tangent space of the Sphere.

### Manifold

We use `Sphere(n-1, k)` from Pymanopt, which represents the set of $(n-1) \times k$ matrices with unit Frobenius norm — equivalent to $\operatorname{vec}(M) \in S^{(n-1)k-1}$.

### Logging

Both optimizers are configured with `log_verbosity=2` to record per-iteration cost values, enabling the required convergence plots.

## Experiments

We run all **12 combinations**: 3 channels × 2 methods × 2 rank values ($k = n-1 = 1023$ and $k = 30$).

## Outputs

1. **Cost vs. iteration plot** for the red channel, comparing SteepestDescent and TrustRegions side-by-side for both $k$ values.
2. **Laplacian extraction** for later use in Part 2: $L^{(c)} = n(Q_\perp M^{(c)})(Q_\perp M^{(c)})^\top$, using the TrustRegions solution with $k = 30$. Sanity checks verify $L\mathbf{1}_n = 0$, $\operatorname{tr}(L) = n$, $L \succeq 0$, and $\operatorname{rank}(L) \leq 30$.
